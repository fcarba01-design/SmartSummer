# ☀️ Verano Sebastián 6º Primaria

Web educativa interactiva creada para preparar el paso a 6º de Primaria.

## Objetivos

- Repasar contenidos de 5º
- Preparar nuevas competencias
- Fomentar autonomía
- Combinar aprendizaje digital y offline


## Asignaturas

- Natural Science
- Ciencias Sociales
- Lengua Castellana y Literatura
- Matemáticas ABN
- Lengua Inglesa
- Music
- Art and Crafts
- Educación Física
- Tecnología y Robótica
- Religión Católica


## Uso

Abrir index.html o publicar mediante GitHub Pages.
